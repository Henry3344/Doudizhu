## ✨ 核心亮点

* **📱 极致轻量**：单文件运行，秒开，不占手机内存。
* **🤖 智能 AI**：内置单机算法，电脑对手会配合、会抢地主、会记牌。
* **🔒 隐私安全**：所有游戏数据（金币、胜率）仅保存在你自己的浏览器本地，**不上传任何服务器**，绝对安全。
* **🎨 自适应 UI**：针对移动端特别优化，支持触屏拖拽选牌。
* **🚫 纯净体验**：模拟“看广告”机制（仅为了增加游戏趣味），实际上**没有任何真实广告**植入。

## 🛠️ 技术栈

* HTML5
* CSS3 (Flexbox/Grid 布局 + 动画特效)
* Vanilla JavaScript (原生 JS，无第三方依赖)
